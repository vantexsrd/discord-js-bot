# Features
- Commands Handler
- Slash Commands Handler
- Context Menu Commands Handler
- Events Handler
- Permission Handler
- Cooldowns

## Installation & Setup
```fix
Start.bat

Leave the ( ServerID ) in ( Source/System.js ) file blank if you want to register slash commands globally.
```
